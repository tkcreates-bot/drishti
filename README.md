
# 🧬 Drishti — AI Medical Report Analyzer

> See your health clearly — no medical degree required.

## What is Drishti?
Drishti is an AI-powered web app that explains medical lab reports in simple English and regional Indian languages. Upload your blood test or lab report PDF and get instant, clear explanations.

## Features
- 📄 PDF report upload & analysis
- 🤖 AI-powered explanation in simple English
- 🌐 Multilingual support (Hindi, Punjabi, Bengali, Telugu, Tamil)
- 💬 Chat with AI about your report
- 🏥 Health score & abnormal value highlights
- 🥗 Diet recommendations based on your report
- 📱 Camera capture for mobile users

## Tech Stack
- **Backend:** Python, Flask
- **AI:** Groq API (LLaMA 3.3)
- **PDF Processing:** PyMuPDF
- **Frontend:** HTML, CSS, JavaScript

## Setup
1. Clone the repo
2. Install dependencies: `pip install -r requirements.txt`
3. Add your Groq API key in `.env` file: `GROQ_API_KEY=your_key`
4. Run: `python app.py`
